Cookmaster

A aplicação trata-se de um cadastro de receitas, onde é possível criar e visualizar receitas, seus ingredientes, e sua forma de preparo. Essa foi a minha primeira aplicação em NodeJs, onde apliquei um CRUD com o padrão de projeto MVC, utilizando a engine EJS para visualização das páginas, banco de dados utilizado MySql.
